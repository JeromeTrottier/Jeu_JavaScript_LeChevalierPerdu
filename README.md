# Le Chevalier Perdu
Créé en JavaScript, CSS et HTML.

## Comment jouer
- Appuyez dans l'affiche du jeu pour démarrer.
- Servez-vous de [W, A, S, D] pour pouvoir vous déplacer.

## But du jeu
Le but est d'accumuler le plus de points (pièces d'or et rubis) sans mourir.
Vous avez 3 vies et donc devez éviter du meilleur de vos capacités les rochers qui pourraient vous tomber sur la tête.

### Détails
Le projet est fait entièrement sur une page HTML et utilise un canvas pour être affiché
